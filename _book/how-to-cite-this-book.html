How to Cite This Book

If you use ideas, structure, figures, or examples from this book in reports, presentations, academic work, or training materials, please cite it as follows.

0.6 Recommended Citation (APA)

Olatunbosun, O. (2025). Power BI for Monitoring and Evaluation and Public Health Data Analysts. https://doi.org/10.5281/zenodo.18178614

Available online at: https://pbi4ds.netlify.app/

0.7 BibTeX Entry

For users of Zotero, Mendeley, or LaTeX, please use the following BibTeX entry:

@book{olatunbosun2025powerbi,
  title = {Power BI for Monitoring and Evaluation and Public Health Data Analysts},
  author = {Olatunbosun, O.},
  year = {2025},
  url = {[https://pbi4ds.netlify.app/](https://pbi4ds.netlify.app/)},
  doi = {10.5281/zenodo.18178614}
}

0.8 Informal Reference

For slide decks or non-academic materials, you may reference the book as:

Olatunbosun, Power BI for M&E and Public Health Data Analysts (2025)

0.9 Versioning Note

This book is published online and may be updated over time.

If citing a specific version, include the access date or version tag where applicable.

Example:

Accessed March 2025.
